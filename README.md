## Problem Description: 
The Network routing optimization problem is determining the most efficient path for
data through a network while minimizing cost and delay. 

This problem can be modeled as a graph where nodes are routers or servers, and the edges 
are communication links that have associated weights such as distance, latency, or cost.

## Algorithms:

- Bellman-Ford
- A*
- (Planned) ACO
- (Planned) GWO
